# BrickBreaker

Canvas と JavaScript で作成した、シンプルなブロック崩しです。

## 遊び方
- 矢印キーまたは A / D でパドルを動かします。
- スペースキーでゲームを開始します。
- ボールを打ち返して、上のブロックを全部崩すとクリアです。
- ボールを落とすと、もう一度始められます。

## ルール
- パドルでボールを跳ね返します。
- ブロックに当たると得点が加算されます。
- すべてのブロックを崩すとクリアです。

## ファイル構成
- index.html: ゲーム画面の構造
- styles.css: 画面の見た目
- script.js: ゲームの動作ロジック

## 起動方法
ローカルで確認する場合は、フォルダを開いて次のファイルを直接表示してください。

```text
BrickBreaker/index.html
```
