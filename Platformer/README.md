# 横スクロールアクション

Canvas と JavaScript で作成した、少しだけ完成度の高い横スクロールアクションのひな型です。

## 遊び方
- 矢印キーまたは W / A / S / D で移動します。
- スペースキーでジャンプします。
- 足場を飛び移り、コインを集めながらゴールを目指します。
- 敵に当たると失敗します。

## 仕様
- 左右移動とジャンプ
- 足場や地面との当たり判定
- 敵の往復移動
- コインの取得
- ゴール判定
- リスタートボタン
- 画面に合わせたカメラ追従

## ファイル構成
- index.html: ゲーム画面の構造
- styles.css: 画面の見た目
- script.js: ゲームの動作ロジック

## 起動方法
ローカルで確認する場合は、次のファイルを直接開いてください。

```text
Platformer/index.html
```
