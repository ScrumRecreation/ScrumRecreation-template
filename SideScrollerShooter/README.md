# 横スクロールシューティング

Canvas と JavaScript で作成した、最小構成の横スクロールシューティングゲームのひな型です。

## 遊び方
- 矢印キーまたは W / A / S / D でプレイヤーを動かします。
- スペースキーで弾を発射します。
- 敵に弾を当てると得点が加算されます。

## ルール
- プレイヤーを操作して敵を倒します。
- 敵は画面右から左に向かって移動します。
- 弾と敵が当たると、両方が消えます。

## ファイル構成
- index.html: ゲーム画面の構造
- styles.css: 画面の見た目
- script.js: ゲームの動作ロジック

## 起動方法
ローカルで確認する場合は、フォルダを開いて次のファイルを直接表示してください。

```text
SideScrollerShooter/index.html
```
