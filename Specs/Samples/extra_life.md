# 仕様サンプル：ハートを取るとライフ＋1

* **関連ISSUE ID:** #103

* **作る人（名前）:** みゆ

* **日付:** 2026-07-15

1) いつ動く？（TRIGGER）
- 記入欄：
```
特定のブロックを壊したときにハートが落ち、プレイヤーのパドルで受け取ったとき
```

2) 何をする？（ACTION）
- 記入欄：
```
残りライフが+1される。ハートは画面外に落ちると消える。
```

3) どんな条件がある？（CONDITION）
- 記入欄：
```
ライフは最大5まで。
ライフが満タンのときはハートを消すのみで効果なし。
```

4) どう見える？（見た目／UI）
- 記入欄：
```
パドルにハートがぶつかったら画面左上のライフ表示が+1される。
```

5) プレイヤーの操作（INPUT）
- 記入欄：
```
パドルでハートを受け取るためにパドルを操作する（矢印キー）
```

6) できたとみなす条件（Acceptance Criteria）
- 記入欄：
```
ハートをパドルで受け取るとライフが1増え、最大5を超えない。
受け取らず画面外に落ちると何も起きない。
```

7) テスト手順（誰でもできる）
- 記入欄：
```
1. 特定ブロックを壊してハートを出す
2. パドルでハートを受け取る
3. ライフ表示が1増えていることを確認する
```

8) 備考（メモ）
- 記入欄：
```
ハートは出現確率20%で落ちる仕様を推奨（調整可）。
```
