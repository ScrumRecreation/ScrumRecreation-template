1) いつ動く？（TRIGGER）
- 記入欄：
```
決まったブロックをこわしたときにハートが落ちて、パドルで取ったとき
```

2) ゲームの中で何が起きる？（ACTION）
- 記入欄：
```
残りライフが+1になる。ハートが画面の外に落ちたら消える。
```

3) どんな条件がある？（CONDITION）
- 記入欄：
```
ライフは最大5まで。
ライフが5のときは、ハートを取ってもライフはふえない。
```

4) 画面に何が表示される？（表示／UI）
- 記入欄：
```
パドルにハートがぶつかったら画面左上のライフ表示が+1される。
```

5) プレイヤーの操作（INPUT）
- 記入欄：
```
ハートを取るために、矢印キーでパドルを動かす
```

6) できたとみなす条件（Acceptance Criteria）
- 記入欄：
```
ハートをパドルで取るとライフが1ふえて、5をこえない。
取らずに画面の外へ落ちたら、何も起きない。
```
