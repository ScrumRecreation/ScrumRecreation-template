# 仕様サンプル：時間イベントでボール速度アップ

* **関連ISSUE ID:** #102

* **作る人（名前）:** けんた

* **日付:** 2026-07-15

1) いつ動く？（TRIGGER）
- 記入欄：
```
ゲームを始めてから20秒ごとにイベントを始める
```

2) ゲームの中で何が起きる？（ACTION）
- 記入欄：
```
イベントが始まってから5秒だけ、ボールの速さを1.25倍にする。
```

3) どんな条件がある？（CONDITION）
- 記入欄：
```
もうイベント中なら、もう一度は始めない。
ゲームオーバー中は始めない。
```

4) 画面に何が表示される？（表示／UI）
- 記入欄：
```
画面上部に「SPEED UP! 5s」と表示する。
```

5) プレイヤーの操作（INPUT）
- 記入欄：
```
なし（自動で発生）
```

6) できたとみなす条件（Acceptance Criteria）
- 記入欄：
```
20秒ごとに5秒間、速度が1.25倍になること。
イベントが終わったら、必ずふつうの速さに戻ること。
```

7) 備考（メモ）
- 記入欄：
```
config.js に eventIntervalSec と eventDurationSec を追加して使う。
```
