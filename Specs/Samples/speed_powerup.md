# 仕様サンプル：スペシャルブロックでボール速くする

* **関連ISSUE ID:** #102

* **作る人（名前）:** けんた

* **日付:** 2026-07-15

1) いつ動く？（TRIGGER）
- 記入欄：
```
ボールが赤い（スペシャル）ブロックに当たったとき
```

2) ゲームの中で何が起きる？（ACTION）
- 記入欄：
```
ボールの速度が一時的に1.5倍になる（5秒間）。
```

3) どんな条件がある？（CONDITION）
- 記入欄：
```
既にスピードアップ中は効果を重ねない。
ゲームの難易度設定が「やさしい」のときは無効。
```

4) 画面に何が表示される？（表示／UI）
- 記入欄：
```
画面中央下に小さなアイコンで「Speed×1.5（5s）」と表示。
```

5) プレイヤーの操作（INPUT）
- 記入欄：
```
なし（自動で発生）
```

6) できたとみなす条件（Acceptance Criteria）
- 記入欄：
```
スペシャルブロックに当たるとボール速度が1.5倍になり、5秒後に元に戻る。
重複効果が発生しない。
```

7) 備考（メモ）
- 記入欄：
```
速度変更はボールの速度ベクトルに対してスカラー倍するだけにする。
```
